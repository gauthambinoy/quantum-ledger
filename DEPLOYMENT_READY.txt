🚀 HOW TO DEPLOY
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Get AWS Credentials
  1. Go: https://aws.amazon.com
  2. Create free account
  3. Create IAM user "terraform-deploy"
  4. Create access key
  5. Download CSV (SAVE IT!)
  Time: 5-10 minutes

STEP 2: Install Terraform
  Mac:   brew install terraform
  Linux: sudo apt install terraform
  Time: 3 minutes

STEP 3: Configure AWS
  mkdir -p ~/.aws
  nano ~/.aws/credentials
  (Paste from CSV)
  Time: 2 minutes

STEP 4: Deploy with Terraform
  cd /home/gautham/cryptostock-pro/terraform
  cp terraform.tfvars.example terraform.tfvars
  nano terraform.tfvars  (edit db_password)
  terraform init
  terraform plan
  terraform apply  (type: yes)
  Time: 15 minutes

STEP 5: Test Your App
  curl http://YOUR-IP:8000/health
  Open browser: http://YOUR-IP:8000
  Time: 5 minutes

TOTAL TIME: ~1 hour to production! ⚡
