## Quick Commands

```bash
# Install & Run
cd mobile
npm install
npm start
npm run ios        # iOS simulator
npm run android    # Android emulator
npm run web        # Web preview

# Development
npm run lint       # Check code style
npm test           # Run tests

# Deployment
eas build --platform ios --profile production
eas build --platform android --profile production
eas submit --platform ios --latest
eas submit --platform android --latest
```

## Common Questions & Answers

**Q: Where do I start?**
A: Read QUICKSTART.md, then run `npm install && npm start`

**Q: How do I deploy to App Store?**
A: Follow DEPLOYMENT.md - iOS Deployment section

**Q: How does it connect to the backend?**
A: See MOBILE_INTEGRATION.md - Authentication Flow

**Q: How can I add a new feature?**
A: 1) Add screen in `src/screens/`
   2) Add service in `src/services/`
   3) Add navigation route in `RootNavigator.tsx`

**Q: Where can I change colors?**
A: Edit `src/utils/themes.ts` (lightTheme/darkTheme)

**Q: How do I test offline?**
A: Enable airplane mode, verify cached data displays

**Q: What are all the screens?**
A: See MOBILE_APP_SUMMARY.md - Screens Breakdown

**Q: How many files are there?**
A: 37 files total - see MOBILE_FILES_MANIFEST.md
